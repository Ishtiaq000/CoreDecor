# CoreDecor
This is an E-commerce web Application for the Furniture Shop

### To Run this app, you need to install all the libraries in the front end and for the backend
#### First, you need to change the directory with the following command
__cd CoreDecor_Frontend__
and then install all the front-end libraries with the following command
__npm i -f__

and You need to open the split terminal to install all the libraries for the backend so go to the backend directory with the following command
__cd backend__
and run the following command
__pip install -r requirement.txt__


after installing all the libraries for the frontend and Backend, you need to run both the frontend and backend 

__Frontend: npm start__
__Backend: python manage.py runserver__
